## 📱 Mobile Applications

### MyXL
A comprehensive mobile app for XL Axiata customers featuring:
- **Tech Stack**: React Native, Redux, Node.js, MongoDB
- **Features**: Bill management, data packages, customer support, loyalty rewards
- **Role**: Full-stack development with focus on user experience and performance optimization

### Paxel
Logistics and delivery platform with real-time tracking:
- **Tech Stack**: Flutter, Firebase, Google Maps API, Node.js
- **Features**: Package tracking, delivery scheduling, driver management, payment integration
- **Role**: Mobile development lead, implementing complex location services and real-time updates

### U by Prodia
Healthcare appointment booking and medical record management:
- **Tech Stack**: React Native, GraphQL, PostgreSQL, AWS
- **Features**: Doctor booking, medical history, lab results, telemedicine consultations
- **Role**: Frontend development with emphasis on accessibility and HIPAA compliance

## 🎨 Design Philosophy

- **Minimalist Approach**: Clean, uncluttered design focusing on content
- **Responsive Design**: Mobile-first approach with seamless desktop experience
- **Subtle Animations**: Purposeful micro-interactions that enhance user experience
- **Typography Hierarchy**: Clear visual hierarchy using custom grading system
- **Accessibility First**: WCAG compliant with keyboard navigation and screen reader support

## 🛠 Technical Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)

- **Performance**: Optimized assets, lazy loading, minimal dependencies
